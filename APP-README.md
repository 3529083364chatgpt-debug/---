# 室友专属赛博回忆录

一个用于记录宿舍和室友回忆的线上纪念簿应用。

## 在线访问

**直接访问应用**：https://3529083364chatgpt-debug.github.io/---/

> 点击上方链接即可直接使用，无需下载或安装。

## 功能特性

### 核心功能（P0）
- **记忆素材录入**：支持照片、聊天记录、文字故事三种类型
- **时间线首页**：按时间倒序展示所有回忆
- **三大内容分类**：照片专区、聊天记录专区、文字故事专区
- **分享功能**：生成分享链接，支持整本回忆录或单条记忆分享

### 增强功能（P1）
- **人物标签与归属**：给每条记忆打上室友标签，可按人物筛选
- **留言互动**：室友可在任意一条记忆下留言评论
- **那年今日**：自动聚合一年/多年前的今天的回忆
- **搜索与多维筛选**：按关键词、日期范围、人物、情绪标签筛选
- **情绪标签**：支持开心、感动、搞笑、怀念、温暖五种情绪

### 设计风格
- 温柔治愈、简约干净、青春校园氛围感
- 暖色调为主（暖白底 #FAF7F2、暖橘强调 #E89B6C、薄荷绿 #7BA69C）
- 移动端优先响应式设计
- 圆角、留白与暖色设计

## 数据存储

应用使用浏览器的 localStorage 进行数据存储：
- `roommate-memory-data`：存储所有回忆数据
- `roommate-memory-comments`：存储留言数据

**注意**：数据存储在本地浏览器中，清除浏览器数据会导致数据丢失。

## 使用指南

### 添加回忆
1. 点击右下角的 "+" 按钮
2. 选择回忆类型（照片、聊天、故事）
3. 填写相关信息并保存

### 浏览回忆
- **时间线视图**：默认按时间倒序展示所有回忆
- **分类浏览**：点击顶部标签切换照片、聊天、故事专区
- **搜索筛选**：使用搜索框或筛选面板快速定位回忆

### 那年今日
- 首页顶部会自动展示历史上的今天的回忆
- 点击卡片可查看详细内容

### 留言互动
- 打开任意回忆详情页
- 在底部留言区输入内容并发送

### 分享回忆
- 点击导航栏的分享按钮
- 选择分享整本回忆录或单条回忆
- 复制链接发送给室友

## 技术实现

- **前端**：纯 HTML + CSS + JavaScript，无框架依赖
- **存储**：localStorage 本地存储
- **响应式**：CSS Media Queries 实现移动端适配
- **图标**：Font Awesome 图标库

## 浏览器兼容性

支持所有现代浏览器：
- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## 注意事项

1. 数据仅存储在本地浏览器中，建议定期备份重要回忆
2. 图片上传会转换为 Base64 格式存储，大量图片可能影响性能
3. 分享链接需要在同一浏览器中访问才能看到数据
