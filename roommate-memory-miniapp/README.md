# 室友专属赛博回忆录 - 微信小程序版本

这是一个用于记录宿舍和室友回忆的微信小程序应用。

## 功能特性

### 核心功能
- **时间线首页**：按时间倒序展示所有回忆
- **三大内容分类**：照片专区、聊天记录专区、文字故事专区
- **记忆素材录入**：支持照片、聊天记录、文字故事三种类型
- **分类筛选**：按类型、时间、情绪等多维度筛选
- **那年今日**：自动展示历史上的今天的回忆

### 详细功能
- **照片回忆**：支持多张照片上传，可添加描述和标签
- **聊天记录**：模拟真实聊天界面，支持多人对话
- **文字故事**：支持长文本编辑，记录完整故事
- **人物标签**：给每条记忆打上室友标签
- **情绪标签**：支持开心、感动、搞笑、怀念、温暖五种情绪
- **留言互动**：每条回忆下可留言评论
- **分享功能**：支持分享给微信好友

## 快速开始

### 1. 环境准备
1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 注册微信小程序账号（可使用测试号）
3. 获取 AppID（在微信公众平台获取）

### 2. 项目配置
1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择本项目目录
4. 填写 AppID（可使用测试号）
5. 点击"导入"

### 3. 运行项目
1. 在微信开发者工具中点击"编译"按钮
2. 使用微信扫码预览
3. 或点击"预览"生成临时二维码

## 数据存储

### 本地存储
- 使用微信小程序的 `wx.setStorageSync` 进行本地存储
- 数据存储在用户设备上，每个用户独立
- 存储键：
  - `roommate_memories`：回忆数据
  - `roommate_comments`：评论数据

### 数据结构
```javascript
// 回忆数据结构
{
  id: "唯一ID",
  type: "photo|chat|story",  // 类型
  title: "标题",
  date: "ISO日期字符串",
  description: "描述",
  content: "故事内容",
  images: ["图片路径数组"],
  messages: [  // 聊天消息
    { speaker: "发言人", content: "消息内容" }
  ],
  persons: ["人物标签数组"],
  mood: "happy|touched|funny|miss|warm",
  createdAt: "创建时间",
  createdBy: "创建者"
}

// 评论数据结构
{
  id: "唯一ID",
  author: "作者",
  content: "评论内容",
  time: "评论时间"
}
```

## 使用指南

### 添加回忆
1. 点击底部"添加"标签
2. 选择回忆类型（照片、聊天、故事）
3. 填写相关信息
4. 点击"保存回忆"

### 浏览回忆
1. **首页**：查看时间线，按时间倒序展示
2. **分类**：按类型筛选回忆
3. **搜索**：在首页搜索框中输入关键词

### 那年今日
- 首页顶部会自动展示历史上的今天的回忆
- 点击卡片可查看详细内容

### 留言互动
1. 打开任意回忆详情页
2. 在底部留言区输入内容
3. 点击"发送"按钮

### 分享回忆
1. 打开回忆详情页
2. 点击"分享"按钮
3. 选择分享给朋友或朋友圈

## 自定义配置

### 修改 AppID
编辑 `project.config.json` 文件，修改 `appid` 字段：
```json
{
  "appid": "您的AppID"
}
```

### 修改样式
- 全局样式：编辑 `app.wxss`
- 页面样式：编辑对应页面的 `.wxss` 文件
- 设计变量：在 `app.wxss` 中修改 CSS 变量

### 添加新页面
1. 在 `pages` 目录下创建新目录
2. 创建四个文件：`.js`、`.json`、`.wxml`、`.wxss`
3. 在 `app.json` 的 `pages` 数组中添加路径

## 扩展功能

### 添加云开发支持
如需多人共享数据，可集成微信云开发：
1. 在 `app.js` 中初始化云开发
2. 使用云数据库存储数据
3. 实现用户认证和数据同步

### 添加图片压缩
可使用微信小程序的图片处理 API：
```javascript
wx.compressImage({
  src: '图片路径',
  quality: 80,
  success: (res) => {
    // 使用压缩后的图片
  }
})
```

### 添加数据导出
可使用 `wx.getFileSystemManager()` 导出数据：
```javascript
const fs = wx.getFileSystemManager()
fs.writeFileSync('文件路径', '数据内容', 'utf8')
```

## 注意事项

### 数据安全
- 本地存储的数据不会自动同步到云端
- 建议定期备份重要数据
- 清除小程序缓存会删除所有数据

### 性能优化
- 图片建议压缩后再上传
- 避免存储过大的图片文件
- 定期清理不需要的回忆

### 兼容性
- 支持微信客户端 7.0 及以上版本
- 支持 iOS 和 Android 平台
- 建议使用最新版本的微信开发者工具

## 常见问题

### Q: 如何备份数据？
A: 目前版本仅支持本地存储，建议截图或手动记录重要回忆。后续版本将添加云同步功能。

### Q: 多个室友如何共享数据？
A: 当前版本为本地存储，每人独立使用。如需多人共享，请集成微信云开发。

### Q: 如何修改头像和昵称？
A: 在 `app.js` 的 `globalData` 中修改 `userInfo` 对象。

### Q: 图片上传失败怎么办？
A: 检查网络连接，确保小程序有存储权限。可尝试压缩图片后重新上传。
