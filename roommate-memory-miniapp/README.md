# 室友回忆录小程序 - 云开发版本

## 🌟 多人共享数据

本版本使用**微信云开发（CloudBase）**，所有室友共享同一份数据，每个人添加的回忆、留言都能被其他室友看到。

## 📋 使用前必读：云开发配置

### 第一步：开通云开发

1. 打开微信开发者工具，导入本项目
2. 点击工具栏上的「云开发」按钮
3. 开通云开发服务，创建一个云环境（名称随意，如 `roommate-memory`）
4. 记住你的**云环境ID**（格式如 `cloud1-xxxx`），在云开发控制台 → 设置 → 环境ID 中查看

### 第二步：替换云环境ID

打开 `app.js`，找到第 9 行：

```javascript
env: 'your-cloud-env-id', // ⚠️ 请替换为你的云环境ID
```

将 `your-cloud-env-id` 替换为你的实际云环境ID。

### 第三步：创建数据库集合

在云开发控制台 → 数据库，创建以下两个集合：

| 集合名称 | 说明 |
|---------|------|
| `memories` | 存储回忆数据 |
| `comments` | 存储留言评论数据 |

### 第四步：设置数据权限

为了让所有室友都能读写同一份数据，需要设置集合权限：

1. 点击每个集合 → 权限设置
2. 选择**「所有用户可读，仅创建者可读写」**（推荐）或**「所有用户可读写」**（更开放）
3. 如果选择前者，室友可以看到所有人的回忆，但只有创建者能修改/删除自己的回忆

### 第五步：替换小程序 AppID

`project.config.json` 中的 `appid` 目前是占位符 `wx1234567890abcdef`，需要替换为你自己的小程序 AppID：

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)
2. 开发管理 → 开发设置 → AppID
3. 替换 `project.config.json` 中的 appid 字段

## 📊 数据结构说明

### memories 集合

```json
{
  "_id": "自动生成",
  "_openid": "自动生成（创建者标识）",
  "type": "photo | chat | story",
  "title": "回忆标题",
  "date": "日期",
  "description": "描述",
  "content": "故事内容（type=story时）",
  "images": ["云存储文件ID列表"],
  "messages": [{"speaker": "发言人", "content": "消息内容"}],
  "persons": ["人物标签"],
  "mood": "happy | touched | funny | miss | warm",
  "createdAt": "创建时间（服务器时间）",
  "updatedAt": "更新时间（服务器时间）"
}
```

### comments 集合

```json
{
  "_id": "自动生成",
  "_openid": "自动生成",
  "memoryId": "关联的回忆ID",
  "author": "留言人昵称",
  "avatarUrl": "头像URL",
  "content": "留言内容",
  "time": "留言时间（服务器时间）"
}
```

## 🔄 与本地版本的区别

- 本地版本（`app-local.js`）：数据存储在 `wx.setStorageSync`，每个人独立，无法共享
- 云开发版本（`app.js`）：数据存储在云数据库，所有室友共享，实时同步

如需切换回本地版本，将 `app-local.js` 重命名为 `app.js` 即可。

## ⚠️ 注意事项

1. 云开发需要在**真机调试**或**预览模式**下才能正常工作，模拟器可能无法使用云能力
2. 图片上传使用了云存储（`wx.cloud.uploadFile`），图片会保存到云端
3. 首次使用时需要用户授权获取昵称和头像（用于留言功能）
4. 基础库版本需 >= 2.2.3 才能使用云能力
